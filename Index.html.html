HTML - HyperText Markup Language

Introdução à Computação - Fundamentos de HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas web. Foi criada por Tim Berners-Lee em 1990 e é a base de todas as páginas da internet. HTML não é uma linguagem de programação, mas uma linguagem de marcação que estrutura conteúdo usando elementos e tags.

Estrutura Básica do HTML

Todo documento HTML possui uma estrutura padrão que deve ser seguida. A estrutura começa com a declaração DOCTYPE, seguida pelos elementos html, head e body. Cada parte tem uma função específica na organização do conteúdo da página.

Exemplo de Estrutura HTML

  • <!DOCTYPE html>
  • <html lang="pt-br">
  • <head>
  • <meta charset="UTF-8">
  • <title>Minha Página</title>
  • </head>
  • <body>
  • <h1>Título Principal</h1>
  • <p>Este é um parágrafo.</p>
  • </body>
  • </html>

Tags para Imagens

A tag <img> é usada para inserir imagens em páginas HTML. É uma tag vazia (self-closing) que requer pelo menos o atributo src (source) para especificar o caminho da imagem. O atributo alt é importante para acessibilidade e o title fornece informações adicionais.

Atributos Especiais para Tabelas

As tabelas HTML possuem atributos especiais que permitem que células ocupem mais de uma coluna ou linha. Os atributos colspan e rowspan são fundamentais para criar layouts de tabela mais complexos.

Versões do HTML

HTML passou por várias versões desde sua criação. Cada versão trouxe novos recursos e melhorias. Atualmente usamos HTML5, que é a versão mais recente e moderna da linguagem.

Evolução das Versões

  1. HTML 1.0 (1991) - Primeira versão básica
  2. HTML 2.0 (1995) - Primeira versão padronizada oficialmente
  3. HTML 3.2 (1997) - Introduziu tabelas e scripts
  4. HTML 4.01 (1999) - Separação de conteúdo e apresentação
  5. XHTML 1.0 (2000) - HTML reformulado como XML
  6. HTML5 (2014) - Versão atual com elementos semânticos

Links e Navegação

A tag <a> (anchor) é usada para criar hiperlinks em HTML. Esta é uma das características mais importantes da web, permitindo a navegação entre diferentes páginas e recursos. Os links podem apontar para outras páginas, arquivos ou seções da mesma página.

Tipos de Links

  • Links Externos - Para outros sites (href="http://exemplo.com")
  • Links Internos - Para outras páginas do mesmo site
  • Links para Email - Para abrir o cliente de email (href="mailto:")
  • Links para Seções - Para partes da mesma página (href="#secao")
  • Links para Arquivos - Para download de documentos
