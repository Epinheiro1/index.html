Pokémon - Primeira Geração

Red, Blue , Yellow e Green (1996-1998)

Sobre os Jogos

A primeira geração de Pokémon foi lançada no Japão em 1996 com os jogos Pokémon Red e Green para Game Boy. Posteriormente, foram lançadas as versões Blue e Yellow, sendo esta última baseada no anime. Estes jogos revolucionaram os RPGs portáteis e criaram uma das maiores franquias de entretenimento do mundo.

Características principais

  • Total de 151 Pokémon diferentes para capturar
  • Região de Kanto como cenário principal
  • Sistema de tipos e batalhas por turnos
  • Conectividade via cabo link para trocas e batalhas
  • 8 Líderes de Ginásio para derrotar

Curiosidades

  1. O jogo foi criado por Satoshi Tajiri inspirado em sua paixão por colecionar insetos
  2. Originalmente, eram apenas 190 Pokémon planejados, mas foram reduzidos para 150
  3. Mew foi adicionado secretamente e só foi revelado após o lançamento
  4. Os jogos venderam mais de 31 milhões de cópias mundialmente
  5. O nome "Pokémon" é uma contração de "Pocket Monsters"

Impacto Cultural

A primeira geração estabeleceu a base para uma franquia que se tornaria um fenômeno cultural global. O conceito de "Gotta Catch 'Em All!" (Tem que pegar todos!) incentivou os jogadores a completar a Pokédex, enquanto o sistema de trocas promoveu a interação social entre os jogadores.

Os jogos também foram fundamentais para popularizar o Game Boy e demonstrar o potencial dos jogos portáteis para experiências profundas e duradouras.
